Map-Based Tower Placement

  Click anywhere on the map to add a tower

  Each tower displays coordinates

  Editable frequency (default: 5 GHz)
RF Link Creation

  Select two towers to create a point-to-point link

  Links require matching frequencies

  Prevents duplicate links automatically

Fresnel Zone Visualization

  Shows the 1st Fresnel zone as an aligned ellipse
